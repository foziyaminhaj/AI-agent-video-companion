# face-api.js Models Directory

This directory should contain the face-api.js model files for emotion detection.

## Required Files

Download the following files from the face-api.js repository:

1. **tiny_face_detector_model-weights_manifest.json**
2. **tiny_face_detector_model-shard1**
3. **face_expression_model-weights_manifest.json**
4. **face_expression_model-shard1**

## Download Instructions

See `AI_FEATURES_SETUP.md` in the project root for detailed download instructions.

## Quick Download Links

- Models repository: https://github.com/justadudewhohacks/face-api.js/tree/master/weights
- Direct download: Use the links provided in `AI_FEATURES_SETUP.md`

## Verification

After downloading, verify the directory contains exactly 4 files listed above.
